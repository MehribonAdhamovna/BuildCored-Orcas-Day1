# BuildCored-Orcas-Day1
# RockLook — BUILDCORED ORCAS Day 01

## What it does

Computer determines whether to play the Rock music or no according to the motion of the user(looking up and down)

## Hardware concept

The process is math integrated : using mirror, system can detect nose and iris location (their coordinates). After determining whether they are up or down, the code runs.
